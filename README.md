# Task Board Frontend

## Setup
1. Install Node.js (>=18)
2. Run:
```
npm install
npm run dev
```

## Features
- Static login with remember me
- Protected routes
- Drag & drop task board (Todo, Doing, Done)
- Create/Edit/Delete tasks
- Search, filter, sort
- Activity log
- LocalStorage persistence
- Reset board
